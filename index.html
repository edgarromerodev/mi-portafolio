El proyecto de Lista de Tareas es una aplicación web que permite a los usuarios gestionar sus tareas diarias de manera eficiente. Con esta aplicación, los usuarios pueden agregar nuevas tareas, editarlas, marcarlas como completadas y eliminarlas cuando sea necesario.
